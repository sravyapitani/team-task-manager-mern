# Team Task Manager - MERN Stack Project

## Project Overview

--> Team Task Manager is a full-stack MERN application developed to help users manage projects and tasks efficiently. The application allows users to register, log in securely, create projects, add tasks, update task status, and manage their work in an organized way.
--> Team Task Manager is a full-stack MERN application that helps users manage projects and tasks efficiently with secure authentication, CRUD operations, MongoDB integration, and a responsive React-based user interface.


# Features

* User Signup and Login Authentication
* Create and Manage Projects
* Create and Manage Tasks
* Update Task Status
* Delete Tasks
* Responsive User Interface
* Secure Backend API
* MongoDB Database Integration

# Installation Steps

## Clone Repository

```bash
git clone https://github.com/sravyapitani/team-task-manager-mern.git
```

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

# Backend Setup

```bash
cd backend
npm install
npm run dev
```


# Environment Variables

Create a `.env` file inside backend folder and add:

```env
MONGO_URI = mongodb+srv://admin:admin123@cluster0.nudi4jm.mongodb.net/taskmanager?appName=Cluster0
JWT_SECRET=your_secret_key
```
